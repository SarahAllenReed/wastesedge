Welcome to Adonthell - Waste's Edge

Install Adonthell

Simply drag the Adonthell package to a location of your choice.

Important note for MacOS X 10.2 (Jaguar) users:
You will have to install the Python 2.3 Framework in order to get Adonthell - Waste's Edge working. This can be downloaded from the Python website.
 

Uninstall Adonthell

Likewise, dragging Adonthell onto the trash bin will completely remove it from your system.
 

Note

If you launch Adonthell directly from this disk image, you won't be able to save your game (as the disk image is write protected).

For more information regarding this software, please check the ReadMe and Playing files.
